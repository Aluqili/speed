# تنبيهات بوت تيليجرام للأدمن

تمت إضافة تكامل Telegram Bot داخل Cloud Functions لإرسال تنبيهات عند الحالات التالية:

1. رسالة دعم جديدة من العميل أو المندوب أو المتجر.
2. إيصال دفع جديد يحتاج مراجعة في الأدمن.
3. طلب شحن محفظة جديد يحتاج مراجعة.

## متغيرات البيئة المطلوبة

قبل نشر Functions، أضف ملفًا محليًا غير مرفوع إلى Git باسم:

`functions/.env`

ثم ضع فيه القيم التالية:

```env
TELEGRAM_BOT_TOKEN=ضع_توكن_البوت_هنا
TELEGRAM_CHAT_IDS=ضع_chat_id_هنا
```

إذا أردت أكثر من جهة استقبال، استخدم عدة Chat IDs مفصولة بفاصلة:

```env
TELEGRAM_BOT_TOKEN=123456:ABCDEF...
TELEGRAM_CHAT_IDS=123456789,-1001234567890
```

## كيف تحصل على Bot Token

1. افتح BotFather داخل Telegram.
2. أرسل `/newbot`.
3. اختر اسم البوت واسم المستخدم.
4. انسخ `Bot Token` الناتج.

## كيف تحصل على Chat ID

### لمحادثة خاصة

1. افتح محادثة مع البوت الذي أنشأته.
2. أرسل له أي رسالة مثل `hi`.
3. افتح هذا الرابط في المتصفح بعد استبدال التوكن:

```text
https://api.telegram.org/bot<TELEGRAM_BOT_TOKEN>/getUpdates
```

4. ابحث عن `chat.id`.

### لمجموعة أو قناة

1. أضف البوت إلى المجموعة أو القناة.
2. أرسِل رسالة داخلها.
3. استخدم `getUpdates` بنفس الطريقة.
4. انسخ قيمة `chat.id` الخاصة بالمجموعة وغالبًا تبدأ بـ `-100`.

## النشر

بعد وضع القيم الصحيحة في `functions/.env`، نفّذ:

```powershell
firebase deploy --only "functions"
```

## ملاحظات مهمة

1. التنبيهات الجديدة تُرسل من الباكند، لذلك ستصل حتى لو كانت لوحة الأدمن غير مفتوحة.
2. الصوت أو النغمة نفسها ستكون حسب تطبيق Telegram وإعدادات الجهاز.
3. إذا أردت لاحقًا توسيع التنبيهات إلى الاعتمادات أو الطلبات أو أحداث أخرى، يمكن إضافة triggers إضافية بنفس المسار.
